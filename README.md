# Grade Calculator (C++)

## What it does
Takes a score (0–100) and outputs a letter grade (A–F).  
If the input is invalid, it shows an error message.

## Requirements
A C++ compiler (g++ or clang++)

## How to run (Windows, MinGW)
```bash
g++ practicebymself.cpp -o app.exe
```
# Command Prompt:
app.exe

# PowerShell:
.\app.exe

## How to run (macOS / Linux)
```bash
g++ practicebymself.cpp -o app
./app
